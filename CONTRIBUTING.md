# Contributing to Universal AI Skills

We love contributions from the community!

## Guidelines

1. **LLM Agnostic:** Do not include hardcoded references to specific models like "ChatGPT", "Claude", "Gemini", or "Llama". Use terms like "AI assistant" or "AI reasoning agent".
2. **Sanitization:** Ensure no personal identifiable information (PII) like names or emails are hardcoded in the prompts.
3. **Format:** Use Markdown (`.md`) format. Provide a clear `# Title` and a brief description at the top.
4. **No Duplication:** Check existing skills before adding a new one to avoid overlap.
5. **Category:** Place your skill in the appropriate subdirectory under `skills/`.

## Pull Request Process

1. Fork the repository and create a feature branch.
2. Add or modify skills following the guidelines above.
3. Verify your skill loads and works with your target agent.
4. Submit a PR with a clear description of what your skill does.

## Code of Conduct

Be respectful and constructive. This is a community resource.
