# Contributing to Universal AI Skills

We love contributions from the community!

To keep this repository truly universal, please adhere to the following rules:
1. **LLM Agnostic:** Do not include hardcoded references to specific models like "ChatGPT", "Claude", "Gemini", or "Llama". Use terms like "AI assistant" or "AI reasoning agent".
2. **Sanitization:** Ensure no personal identifiable information (PII) like names or emails are hardcoded in the prompts.
3. **Format:** Use Markdown (`.md`) format. Provide a clear `# Title` and a brief description at the top.

Submit a Pull Request, and we'll review it!
